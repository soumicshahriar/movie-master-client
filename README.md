## Features

- Browse all movies with poster, rating, genre, and release year.
- Add, update, or delete your own movies (CRUD operations with authentication).
- View your personal collection and manage watchlist.
- Advanced filtering by genre and rating range.
- Fully responsive design for mobile, tablet, and desktop.
- Night/Dark theme toggle.
- Animated carousels for featured movies.
- Smooth error handling and toast notifications for all actions.

## Technologies Used

- **Frontend:** React, React Router, Tailwind/DaisyUI, Framer Motion , Uiverse.io, sweetalert2, react-toastify 
- **Backend:** Node.js, Express, MongoDB  
- **Authentication:** Firebase
